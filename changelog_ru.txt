0.10.0b [2020-01-17]
+ Доработан дизайн меню.
+ Добавлен таймаут для операций в меню торговли для предотвращения исполнения неавторизованных запросов (запросы синхронизации, покупка/создание и отмена сделок). 
Таймаут индивидуален для каждого игрока и сбрасывается при открытии меню с помощью команды.
+ Добавлен кулдаун в 1 секунду для отправки пакетов - операций управления в меню торговли (создание/отмена сделок).
+ Добавлен кулдаун в 500 миллисекунд для отправки пакетов - операций в меню торговли (покупка).
+ Добавлен запрет на покупку при полном заполнении почтового ящика.
+ Добавлена возможность запрета открытия меню торговли со стороны клиента. Для открытия меню для конкретного игрока может использоваться команда.
+ Доработана глобальная история покупок. Историю своих покупок и продаж игрок может просмотреть в меню торговли.
+ Добавлено меню глобальной истории покупок.
+ Добавлена система расчета привлекательности сделок на основе истории покупок. На основе данных истории вычисляется средняя стоимость
одного предмета и на основе этого значения расчитывается отклонение стоимости в процентах для каждой сделки - процент привлекательности. 
При наведении курсора на процент привлекательности выводится статистика по предмету:
"(количество проведённых транзакций/общее количество проданных предметов): средняя стоимость одной единицы". Процент привлекательности
отображается для сделок в разделах "Покупка" и "Сделки". Для раздела "Покупка" добавлен фильтр привлекательности сделок.
+ Добавлена возможность при продаже указывать стоимость единицы товара в виде десятичного числа с точностью до 2 знаков. Стоимость единицы 
товара теперь отображается в виде десятичного числа.
+ Добавлен вывод средней стоимости единицы предмета для всех предметов в инвентаре в разделе "Продажа".
+ Обновлены параметры конфигурации:
 Клиент 
 - "enable_trade_menu_key" [boolean] - разрешить добавление клавиши "открыть меню торговли"
 
 Сервер 
 - "trade_menu_operations_timeout_millis" - [int] - таймаут операций в меню торговли в миллисекундах
 - "enable_trade_menu_access_clientside" - [boolean](синхронизируется) - разрешить открытие меню торговли со стороны клиента
 - "enable_self_purchase" - [boolean] - разрешить покупку своих вещей
 - "max_offers_per_player" - [int](синхронизируется) - макимальное количество сделок на игрока
 - "items_per_offer_max_amount" - [int](синхронизируется) - максимальное количество предметов, которое можно продать за одну сделку
 - "price_max_value" - [long](синхронизируется) - максимальная стоимость сделки
 - "offer_expire_time_hours" - [int](синхронизируется) - срок размещения сделки
 - "enable_sales_history" - [boolean] - разрешить глобальную историю покупок
 - "enable_sales_history_sync" - [boolean](синхронизируется) - разрешить синхронизацию глобальной истории покупок со всеми
 - "sales_history_expire_time_hours" - [int] - время хранения записи о сделке в истории покупок
 - "offer_creation_fee_percent" - [int](синхронизируется) - денежный сбор за создание сделки (процент от стоимости)
 - "offer_sale_fee_percent" - [int](синхронизируется) - комиссия с продажи (процент от стоимости)
 
+ Добавлены команды:
 Сервер - Операторы [/oxygenop]
 - Открыть меню торговли: /oxygenop trade -open-menu <target>
 - Отменить сделку: /oxygenop trade -cancel-offer <offer id> <flag>
 - Очистить глобальную историю покупок: /oxygenop trade <mode> <args>
 
 Клиент [/oxygenc]
 - Открыть меню торговли: /oxygenc trade
 - Открыть меню глобальной истории покупок: /oxygenc trade -sales-history
 - Сбросить данные клиента: /oxygenc trade -reset-data
 
+ Добавлены привилегии:
 - "trade:maxOffersPerPlayer" [id - 1000, type - BOOLEAN] - макимальное количество сделок на игрока
 - "trade:marketAccess" [id - 1001, type - BOOLEAN] - возможность доступа к меню торговли
 - "trade:salesHistoryAccess" [id - 1002, type - BOOLEAN] - возможность доступа к глобальной истории покупок
 - "trade:tradeMenuOperatorOptions" [id - 1003, type - BOOLEAN] - возможность использовать дополнительные опции для оператора в меню торговли (отмена/удаление сделок)
 - "trade:itemsPerOfferMaxAmount" [id - 1010, type - BOOLEAN] - максимальное количество предметов, которое можно продать за одну сделку
 - "trade:priceMaxValue" [id - 1011, type - BOOLEAN] - максимальная стоимость сделки
 - "trade:offerCreationFeePercent" [id - 1020, type - BOOLEAN] - денежный сбор за создание сделки (процент от стоимости)
 - "trade:offerSaleFeePercent" [id - 1021, type - BOOLEAN] - комиссия с продажи (процент от стоимости)
 
+ Добавлены клиентские настройки:
 Общие

 - использовать систему расчёта привликательности сделок
